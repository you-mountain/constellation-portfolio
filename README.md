## ✨ The Story Behind This Project

Tired of boring, generic portfolio templates that all look the same, I built a **constellation map** that represents my professional journey. Clicking on stars reveals different chapters of my career, all connected like a personal galaxy. The entire project is built with **pure HTML, CSS, and JavaScript**—no frameworks, no bloat.

## ✨ Features

- 🌟 **Interactive Star Map** - Click on stars to explore different sections
- 🎨 **Glassmorphism Design** - Modern frosted glass UI with `backdrop-filter`
- ✨ **Animated Night Sky** - Dynamic background with twinkling stars
- 🔗 **Connected Constellation Lines** - Stars form a unique network
- 💫 **Floating Particle Effects** - Magical particles around modal cards
- 📱 **Fully Responsive** - Works perfectly on mobile, tablet & desktop
- ⚡ **Zero Dependencies** - Pure HTML, CSS, and vanilla JavaScript
- 🌐 **Bilingual Support** - English & Persian versions available
- 🔓 **Open Source** - MIT licensed, free for personal and commercial use

## 🎮 How It Works

1.  **Stars** represent different sections of your portfolio (About, Skills, Projects, Contact, etc.)
2.  **Connections** form a unique constellation pattern
3.  **Clicking a star** opens a beautiful glassmorphism card with details
4.  **Interactive background** creates an immersive experience

## 🛠️ Tech Stack (Zero Dependencies!)

*   **HTML5** - Semantic structure
*   **CSS3** - Glassmorphism, Animations, Grid & Flexbox
*   **JavaScript** - Canvas API, Event Handling, Dynamic Content
*   **No React, No Vue, No Bootstrap!**
